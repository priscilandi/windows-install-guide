KBA: How to Install Windows

Objective

To perform a clean installation of Windows and configure it for the corporate environment, including all necessary updates and domain joining.

Prerequisites

  • Laptop connected to a power source.
  • Windows installation media available.
  • Corporate domain credentials (for final step).
  1. Power on the laptop and press Enter to access the Startup Interrupt Menu.
  2. Press F1 to enter the BIOS Setup Utility.
  3. Navigate to Startup > Boot.
  4. Change the 1st Boot Priority to the correct drive (e.g., hard drive/USB).
  5. Save changes and exit. The laptop will boot into Windows Setup.
  6. Accept the license terms.
  7. Choose "Custom: Install Windows only (advanced)".
  8. Select the largest partition, click Format, then click Next.

After Windows setup completes, attempt to connect to a Wi-Fi network. If networking is unavailable, you must manually install the appropriate Wi-Fi drivers for the laptop model.

  1. Go to Settings > Windows Update.
  2. Click Check for updates and wait for all items to download and install.
  3. Reboot the computer when prompted.
  4. Repeat this process until no more updates are found.
  1. In the Windows search bar, find and open the Lenovo System Update utility.
  2. Scan for new updates.
  3. Apply all recommended and critical updates.
  4. Reboot if required to complete the installations.
  1. Search for and open "Access work or school".
  2. Rename the computer to the standard format: N10-[SerialNumber].
  3. Join the computer to the domain: UNIONAGENER.com.
  4. Provide domain administrator credentials when prompted.
  5. Reboot the computer one final time to apply all changes.

Process Complete

The laptop is now fully configured and ready for use.
